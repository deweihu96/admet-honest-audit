The problem

Leaderboards look precise.
Often they aren't.

TDC ADMET is a standard leaderboard ranking ML models on drug properties — solubility, toxicity, permeability.

A 2026 audit found most top entries fail reproducibility or leak test data — and that overfitting the open test set is a reliable way to climb.

So a high rank may reflect overfitting, not a better model.

What we built

One agentic loop that climbs and audits.

Climb

Improve models

An agent proposes model designs, trains them, and selects among them to improve validation performance.

Audit

Check integrity

The same loop checks every model and data split for leakage, and reports results with honest uncertainty.

The structural guarantee

The part of the system that can see the test set cannot influence the part that designs models — so it can't cheat, by construction. Built as an autonomous research agent with Claude Code.

The discipline · stress-tested
